# Up and Running with Protractor

Protractor is an end-to-end testing library for AngularJS. This tutorial covers the following: 

  - Installation for Mac OSX and Windows
  - Setting up Protractor 
  - Writing a Test

## Installation
NodeJS is a pre-requsite. Install node for Windows [here](http://blog.teamtreehouse.com/install-node-js-npm-windows) and for OSX via [HomeBrew](http://brew.sh/)

Once NodeJS is installed, run the <pre><code>npm install –g protractor</code></pre> command. This will install everything on your machine that you need to run Protractor.

Go to your command line again and type <pre><code>protractor -–version.</code></pre>
If you see a version, it means that node was successfully installed.
